# Conversor de Moedas

Este é um projeto desenvolvido com fins puramente didáticos, focado no aprendizado e prática de desenvolvimento mobile com **Flutter**.

A aplicação consiste em consumir uma API externa para obter dados em tempo real sobre o mercado financeiro. O objetivo é permitir a conversão de moedas estrangeiras (como Dólar, Euro, etc.) para o Real, além de acompanhar outros indicadores econômicos.

##  Objetivos do Projeto

- **Consumo de API**: Praticar a realização de requisições HTTP e o tratamento de respostas JSON.
- **Manipulação de Dados**: Exibir informações atualizadas de cotações financeiras.
- **Interface do Usuário**: Criar uma interface amigável para visualização e conversão de valores.

## Funcionalidades

- Exibição da cotação atual de diversas moedas.
- Conversão de valores em tempo real com base na cotação do momento.
- Visualização de indicadores do mercado financeiro.

## Tecnologias

- [Flutter](https://flutter.dev/)
- [Dart](https://dart.dev/)
- [HG Brasil – Finance API](https://hgbrasil.com/status/finance)


##  Como executar

Para rodar o projeto localmente, certifique-se de ter o ambiente Flutter configurado e execute os seguintes passos:

1. Clone o repositório.
2. Baixe as dependências do projeto:
   ```bash
   flutter pub get
   ```
3. Execute o aplicativo em um emulador ou dispositivo físico:
   ```bash
   flutter run
   ```
